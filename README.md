# elements-viewer

This is an elements viewer based on the https://github.com/PolymerLabs/indie-catalog
## Setup

### 1. Install npm dependencies
```
npm i
```

### 2. Install bower dependencies and run build

```
npm start
```

## 3. Serve

```
polymer serve
```

## Development tasks

### Rebuild the app

```
npm run debug
```

### Download new versions and rebuild the app

```
npm run build
```
